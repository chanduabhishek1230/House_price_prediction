# House Price Prediction

OVERVIEW

This project predicts house prices using Machine Learning based on inputs like location, size, and number of bedrooms.

---

 ⚙️ Technologies

* Python
* Pandas
* Scikit-learn
* Streamlit

---

🚀 How to Run

pip install -r requirements.txt
 streamlit run app.py

---

 📁 Files

* app.py → main app
* rf_model.joblib → trained model
* model_columns.joblib → features
* cleaned_df.csv → dataset
